**Job Recommendation Engine**

A Java-based console application that recommends jobs based on user skills.

**Features**

- Skill-based Job Recommendations
- Match Percentage Calculation
- Skill Gap Analysis
- Learning Recommendations
- Location and Salary Insights
- Ranked Job Listings

**Technologies Used**

- Java
- OOP Concepts
- Collections Framework
- File Handling
- Git & GitHub

**Sample Output**

- Recommends jobs based on user skills
- Shows missing skills
- Suggests learning paths
- Displays location and salary information

**Future Enhancements**

- MySQL Integration
- Java Swing GUI
- Resume Matching
- AI-powered Recommendations
